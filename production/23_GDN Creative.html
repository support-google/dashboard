반응형 광고 이미지 Best Practice - 이미지

업로드형 이미지 형식의 게재위 및

Point

01
    이미지 사이즈   Image Size
    크기 : 가로형 1개(1200x628) & 정사각형 1개(1200x1200)
    용량 : 각 최대1MB

Point

02
    CTA 및 클릭 유도 문안   Call To Action
    지양요소 : 텍스트나 액션유도 문구가 들어간 이미지는 지양
    권장 : 텍스트 활용 시 광고의 20% 이하로, 좌우 5% 여백 필수

Point

03
    이미지 연관성   Image Association
    텍스트 & 랜딩페이지와 연관성 있는 이미지 사용

Point

04
    간결한 이미지 사용   Using a Simplified Image
    모바일 최적화 : 작은 지면을 고려한 간결한 이미지 사용

Point

05
    로고   모바일 / PC 모두 노출
    로고 미업로드 : 이미지에 로고 포함 권장
    로고 업로드 : 이미지에 로고 포함 지양

반응형 광고 이미지 Best Practice - 이미지

업로드형 이미지 형식의 게재위 및

Point

01
    제목 길이   Length of Title
    광고 제목 : 최대 40자까지 (한글 20자) / 최대5개

Point

02
    제목 내용   Title Content
    지양요소 : 단순히 제목을 언급하는 것이 아니라, 사용자의 흥미와 영감을 불러 일으키는 제목 사용
    권장 : 캐로셀 형태에서는 모든 활용 이미지 카드에 통용될 수 있는 제목을 사용

Point

03
    랜딩페이지 명시   Write Landing Page Clearly
    지양요소 : 유저가 랜딩페이지에서 보게 될 내용을 괄호내 명시
    권장 : [템플릿], [팁], [무료 다운로드], [인포그래픽] 등 명시시 CTR + 88%
