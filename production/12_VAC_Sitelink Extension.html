Sitelink Extension 개요

VAC - Sitelink Extension(Add on)

VAC 인스트림 영상 하단에 사이트내 특정 페이지로 이동하는 링크를 함께 노출시키는 추가 기능

랜딩페이지와 관련있는 페이지도 함께 노출됨으로써 광고의 확정성 증대

Sitelink Extension > Best Practice

Best Practice

최적의 성과를 위한 Best Practice

텍스트 문구, 콜투액션, 전환추적, 사이트링크 모두 활용 권장

  • 텍스트 문구   Text Description

    헤드라인은 최대한 간결하게 작성

    설명 문구 추가 옵션 활용


  • 콜투액션   Call-To-Action

    사이트링크와 관련된 메세지 활용

    ex. 추가할인, 무료배송


  • 전환 추적   Conversion Tracking

    각 사이트링크 랜딩페이지 별 전환추적권장

    광고와 상호작용(클릭/조회)한 후의 전환을 추적하기 위해 구글애즈태그 권장


  • 사이트링크   Site Link

    최소 4개 이상의 사이트링크 활용 권장

    캠페인/광고그룹당 모두 활용 가능하며, 최대6개까지 가능
