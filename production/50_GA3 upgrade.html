Process 1

1. 업그레이드 준비

기존 계정 업그레이드 준비 단계

  • Process
    01
    GA3 계정 로그인 
     GA3 계정 로그인 후 좌측 하단 톱니바퀴 클릭
     GA4 속성 항목 중 설정 어시스턴트 클릭
  • Process
    02
    GA4 계정 생성 세팅 
     시작하기 클릭

Process 2

2. GA 계정/속성 설정

계정 세팅 단계

  • Process
    03
    GA 계정 설정 
     1번 : GA4 속성 생성 과정에 대한 요약 안내문. 기존UA 버전 속성에 영향 없음. 기본 설정을 복사해서 GA4 속성을 생성함. 전환 잠재고객, 이벤트, 제품 링크 등의 추가 설정은 해야 함, 향상된 측정 기능은 자동으로 시작됨 등의 내용.
     2번 :UA속성의 스크립트 또는 구글 태그매니저로 구현된 태그를 온전히 사용할 수는 없고 일련의 작업이 필요하다는 뜻
     3번 :속성만들기 클릭

Process 3

3. 전환 추적 스크립트 발급

트레킹 코드 발급 및 하드코딩 단계

  • Process
    04
    데이터 스트림 발급 
     GA4 속성 보기를 클릭한 다음 생성된 화면
     해당 화면이 보이면 GA4 속성 생성이 완료된 것. 이제 중요한 것은 GA4 속성이 웹 데이터를 받을 수 있게 구현하는 작업. 기존에 유니버셜 애널리틱스를 어떻게 구현했는지에 따라 연동작업이 달라지게 됨. 자세한 연동 프로세스는 “GA4 - GA3 연동 프로세스” 파일을 확인
