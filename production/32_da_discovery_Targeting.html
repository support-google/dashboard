맞춤 잠재고객

    맞춤 잠재고객 타겟팅
    페이지뷰 트리거 종류 1
    맞춤 잠재고객 타겟팅 활용   Custom Audience Targeting
  • 맞춤 잠재고객
    주요 기능 : 맞춤 잠재고객은 캠페인 목표 및 키워드를 기반으로 이 모든 신호를 해석하여 도달범위, 구매 고려도, 실적을 높이고 계정을 효율적으로 관리할 수 있도록 하는 기능
  • 주요 요소
    타겟팅 주요 요소 :
         요소 1 : 특정 유형의 웹사이트를 둘러보는 사용자
         요소 2 : 특정 유형의 앱을 사용하는 사용자
         요소 3 : 다음 관심분야 또는 구매 의도가 있는 사용자 또는
         요소 3-1 : Google에서 다음 검색어를 하나라도 검색한 사용자

    조합3과 조합3-1의 차이
    모수를 수집하는 기준에 차이가 있음 (개념상 조합3 옵션이 더 포괄적)

    페이지뷰 트리거 종류 2
    검색한 사용자 / 관심분야 또는 구매의도  Options
  • 검색한 사용자 타겟팅
    주요 기능 : 최근 YouTube와 Google에서 설정한 키워드를 검색한 유저를 타겟팅(Goolge 검색어를 기준으로 사용자를 분류)

    상품별 작동 방식 :
    YouTube, Gmail, Discovery : 검색어를 입력하여 맞춤 잠재고객을 설정 시, 해당 검색어 및 유사한 검색어를 사용한 사용자에게 광고가 게재
    그 외 상품 : Google 서비스(예: Google.com, YouTube) 검색어를 이용하지 않는 캠페인으로 일반 관심분야 또는 구매 의도(첫번째 옵션)활용

  • 관심분야 및 구매의도
    주요 기능 : 관심사 키워드로 Google 서비스의 검색어와는 별개로 해당 사용자의 관심사를 타겟팅

    상품별 작동 방식 : 모든 상품에서 동일하게 작동
