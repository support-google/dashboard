GA4 Creative Account > Setting Process

Process 1

1. Gmail - GA계정 생성

계정 생성 단계

  • Process
    01
    Gmail로그인 후 GA 계정 생성 
     구글애즈에서 “관리“ 권한을 갖고있는 Gmail로 로그인 후 측정 시작 버튼 클릭

Process 2

2. GA 계정/속성 설정

계정 세팅 단계

  • Process
    02
    GA 계정 설정 
     계정이름/계정 데이터 공유 설정(1번/2번)모두 입력
     다음 버튼 클릭(3번)
  • Process
    03
    GA 속성 설정 
     속성설정 (1번) 항목 모두 입력(보고시간대 / 통화 대한민국으로 수정)
     고급 옵션 보기 클릭
  • Process
    04
    고급 옵션 설정 
     우상단 토글버튼 활성화
     추적하고자 하는 페이지의 URL 주소 입력
     GA4와 GA3를 모두 생성할 경우 : Google 애널리틱스4 속성과 유니버셜 애널리틱스 속성 둘 다 만들기 클릭 (해당 가이드는 3번 옵션 선택)
         GA3만 생성할 경우 : 유니버셜 애널리틱스 속성 만들기 클릭
  • Process
    05
    비즈니스 정보 입력 
     업종 카테고리 / 비즈니스 규모 / 계획 입력 (추후 변경 가능합니다.)
     만들기 클릭
  • Process
    06
    동의 항목 
     애널리틱스 서비스 약관 계약 동의함 클릭
     내 이메일 커뮤니케이션 체크박스 선택 후 저장 클릭

Process 3

3. 전환 추적 스크립트 발급

트레킹 코드 발급 및 하드코딩 단계

  • Process
    07
    데이터 스트림 발급 
     좌측 하단 톱니바퀴 모양 클릭(1번)
     속성 중 데이터 스트림 클릭(2번)
  • Process
    08
    데이터 스트림 추가 
     광고주 URL이 입력된 데이터 스트림 옵션창 선택(해당 가이드는 1번 선택)
     앱/웹을 따로 발급받으시는 경우 우측 스트림 추가 클릭 후 옵션 선택
    GA4/GA3 공용 전환 추적 스크립트 발급 
     새로운 온페이지 태그 추가 클릭
     해당 태그를 HTML의 HEAD와 /HEAD 사이, Title태그와 Meta 태그 아래에 하드태킹(광고주측에 안내 필요)
  • Process
    09
    기존 GA3 전환 추적 스크립트 발급 시 
     GA3(UA)버전에서 발급받는 범용 사이트 태그를 심는 경우 GA4에서 올바른 데이터를 받아볼 수 없음
     GA4 버전의 전체 사이트 태그로 진행 필요
