CPM Masthead 단가 및 상품 정책

예약형 상품 가이드
마스트헤드 구매방식 & 최저 집행 조건

Minimum Guarantee / 위약금기간 / 단가 / 정책관련 안내

  • 최소집행금액   Minimum Guarantee

    3,000만원

    단 타겟팅 적용으로 인해 인벤토리 규모가 3천만원 미만일 경우 해당 인벤토리를 100% 예약해야만 3천만원 미만으로 집행가능


  • 위약금 기간   penalty period

    집행일로부터 2주이내

    계약서 동의 시점부터 적용


  • 시작 CPM 단가   least CPM unit Cost

    2,800원부터 시작

    노출빈도 제어 / 지역 타겟팅 / 언어 타겟팅 옵션 포함가


  • 타겟팅당 할증   Premium per Targeting

    타겟팅 옵션당 +700원 할증

    데모타겟팅 / 기기타겟팅 / 자녀유무 타겟팅 / 관심사타겟팅


  • 집행 불가 카테고리   Premium per Targeting

    2021/06 이후부터 선거, 정치, 도박, 알콜, 제약 집행불가

    자세한 규정사항은 구글 담당자에게 문의


CPM Masthead 집행 프로세스

예약형 상품 가이드
CPM 집행 프로세스

상품 부킹부터 집행까지 프로세스 안내
  • Invt Check

    Process 1 : Inventory Check

    -. 간략한 캠페인 예산, 기간, 타겟팅과 함께 인벤토리 확인
    -. 수신되는 예약형 팀의 이메일을 기준으로 상세 캠페인 내역 전달
    -. Cross Check 후 부킹 요청

  • Booking

    Process 2 : Booking Masthead

    -. Masthead 부킹 동의 메일 수신
    -. 동의 메일 검토 후 계약서 확정

  • Setting

    Process 3 : Creative Setting

    -. 프리뷰툴(https://www.youtube.com/videomasthead/)활용하여 소재 세팅
    -. 프리뷰툴 링크를 활용하여 예약형 팀에 소재 전달(라이브 1주전, 최소 4일전)
    -. 광고 승인 메일 확인

  • Activation

    Process 4 : Active and can get Report

    -. 부킹날짜에 맞춰 광고 라이브
    -. 광고 종료 후 리포트 수령
