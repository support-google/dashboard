App Campaign for installs - 에셋의 종류 / 조합 방식

업로드형 이미지 형식의 게재위 및
    광고 에셋  

    이미지(선택) : 최대 20개

        가로모드(1.91:1) : 200X200(최소) 1200x1200(권장) / 최대5MB
        정사각형(1:1) : 600X314(최소) 1200x628(권장) / 최대5MB
        세로모드(4:5) : 320X400(최소) 1200x1500(권장) / 최대5MB

    HTML(선택) : 최대 20개

        소재 : https://support.google.com/google-ads/answer/9948381?hl=ko

    동영상(선택) : 최대 20개

        소재 : Youtube 동영상 URL 삽입

    광고제목(필수) : 2개이상 / 최대 5개

        의미 : 광고 제목은 사용자의 눈길을 광고로 집중시키며 다른 애셋과 함께 사용할 수 있는 단어나 구문
        텍스트 제한 : 각 광고제목은 30자(영문 기준) 이하여야 함
        최소 갯수 : 다른 애셋과 조합할 수 있는 광고 제목을 2개 이상 입력(최대 5개)
        호환성 : 광고 제목이 그 자체로 의미가 있고 다른 에셋과 조합할 수 있는지 검토

    설명(필수) : 1개이상 / 최대 5개

        의미 : 설명은 사용자가 앱과 상호작용하도록 유도하는 단어나 구문이며 다른 애셋과 함께 사용할 수 있음
        텍스트 제한 : 각 설명은 영문 기준 90자 이하여야 함
        최소 객수 : 다른 애셋과 연결할 수 있는 고유한 설명을 1개 이상 입력(최대 5개)
        호환성 : 각 설명이 그 자체로 의미가 있고 다른 애셋과 연결될 수 있는지 검토

전환수 최대화 입찰전략

전환 기반 최적화

Point

01
    상품 목표 : 전환 수  
    목표 : 정해진 예산 내에 전환단 단가(CPA)와 상관없이 가장 효율적인 전환을 확보하는 것이 목표

Point

02
    캠페인 목표 : 전환 수 > 전환 단가  
    전환 확보가 목표이지만 전환 단가보다 전환 볼륨이 중요한 경우

Point

03
    입찰 전략 : 자동  
    머신러닝을 사용하여 입찰가를 자동으로 최적화하고 각 입찰에 따라 입찰가를 맞춤설정하는 실시간 입찰 기능을 제공
    예산 내에서 가장 많은 전환수를 확보하는 로직으로 작동

Point

04
    입찰가 조정 : 불가  
    볼륨중시 : 타겟 CPA를 활용하지 않는 입찰방식이며, 전환수 극대화를 목표로 예산 지출
    가이드라인 설정 불가 : 캠페인 생성 시 각 전환에 대해 지불하고자 하는 평균 비용을 설정 불가

Point

05
    전환추적  
    AOS : 앱 설치(Playstore 기준)
    First Open(타사 앱 분석)
    iOS : First Open(타사 앱 분석)

Point

06
    일별 최소 전환, 예산  
    일 예산 : tCPI X 50 이상
    일 평균 전환 수 : 10회 이상
일반 경매형(Auction Type) 입찰전략
타겟 CPA

전환 기반 최적화

Point

01
    상품 목표 : 전환 수 + 타겟 CPA  
    목표 : 목표로 하는 전환당 단가(CPA)를 유지하면서 예산 내에서 최대한 많은 전환확보가 목표

Point

02
    캠페인 목표 : 전환 수 + 전환 단가  
    전환 확보가 목표이고 이미 인지하고 있는 전환당 단가(CPA)가 있는 경우

Point

03
    입찰 전략 : 자동  
    고객이 광고주의 제품 또는 서비스에 맞는 Google 검색을 할 때, 입찰 광고의 전환 가능성에 따라 타겟 CPA를 사용하여 입찰가를 설정
    일부 CPA가 tCPA보다 높거나 낮을 수 있지만, 설정된 tCPA와 성과가 최대한 같아지도록 조절

Point

04
    입찰가 조정 : 가능  
    단가+볼륨 중시 : 타겟 CPA는 전환 1회에 대해 지불하고자 하는 평균 금액
    가이드라인 설정 : 캠페인 생성 시 각 전환에 대해 지불하고자 하는 평균 비용을 설정

Point

05
    전환추적  
    AOS : 앱 설치(Playstore 기준) / First Open(타사 앱 분석)
    iOS : First Open(타사 앱 분석)

Point

06
    일별 최소 전환, 예산  
    일 예산 : tCPI X 50 이상
    일 평균 전환 수 : 10회 이상
