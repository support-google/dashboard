GTM Creative Account > Setting Process

Process 1

1. GTM 계정 생성

계정 생성 단계

  • Process
    01
    Gmail로그인 후 GTM페이지 진입 
     GTM 계정을 만들고자하는 Gmail로그인 후 첫 화면에서 "계정만들기" 버튼 클릭
     GA에서 수정권한 이상의 권한을 가진 Gmail계정으로 진행하는 것을 권장
  • Process
    02
    세 계정 추가 
     GTM 계정 이름 : 회사/브랜드 명 입력
     국가 : 추적하고자 하는 웹페이지가 소속된 국가 입력
     컨테이너 이름 : 추적하고자 하는 웹페이지의 메인 도메인 입력
     타겟 플랫폼 : 추적하고자 하는 웹페이지가 속하는 플랫폼을 선택
  • Process
    03
    Google 태그 관리자 서비스 이용약관 
     하단 GDPR에서 요구하는 데이터 처리 약관에 동의 체크
     "예" 버튼 클릭 후 다음 페이지로 이동

Process 2

2. GA 컨테이너 태그 삽입

트레킹 코드 발급 및 하드코딩 단계

  • Process
    04
    GTM 컨테이너 태그 삽입 
     1번 태그의 경우 head태그에서 가능한 한 높은 위치(일반적으로 meta태그와 title태그 아래를 의미)에 붙여 넣음
     2번 태그의 경우 body태그 바로 뒤에 붙여 넣음
