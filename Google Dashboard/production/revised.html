Text Creative Guideline

Text Creative Guideline

건너뛸 수 있는 인스트림 광고의 경우 최적의 결과를 얻으려면 30초 이하의 동영상 사용 권장
  • Headline

    Checkpoint 1 : Headlilne

    -. 최대 15자(영문 기준)
     -. 클릭 유도 문구 포함

  • URL

    Checkpoint 2 : Display URL

    -. 최대 255자(영문 기준)

  • CTA

    Checkpoint 3 : Call-to-Action

    -. 최대 10자(영문 기준)

  • Option

    Checkpoint 4 : Creative Option

    -. 제품 피드 세팅 가능
    -. 사이트 링크 광고 확장 세팅 가능
    -. 광고 소재 옵션을 추가하면 광고의 전체적인 모습이 달라질 수 있음. 캠페인별로 한 번에 하나의 광고 소재 옵션만 표시 가능
    -. Mobile Only
